¿Qué es Arduino?
1
Plataforma de hardware y software abierto

Arduino es una plataforma de hardware y software de código abierto que facilita la creación de proyectos electrónicos interactivos. Fue diseñada para ser accesible tanto a principiantes como a profesionales.


Permite conectar sensores, actuadores y otros componentes electrónicos al mundo físico mediante una programación sencilla basada en C/C++.

Plataforma de prototipos electrónicos
Fácil de usar para principiantes
Ideal para proyectos de IoT
Gran comunidad de apoyo
Hardware y software de código abierto
Compatible con cientos de librerías

Componentes Principales
2
Partes esenciales de la placa Arduino UNO

Conoce las partes clave que componen una placa Arduino y su función dentro del sistema:

Microcontrolador ATmega328P
Pines digitales (0–13)
Pines analógicos (A0–A5)
Puerto USB para programación
Alimentación 5V / 3.3V
LED integrado en pin 13
Botón de reset
Pines PWM (~3, ~5, ~6, ~9, ~10, ~11)
IDE
Entorno de Programación (Arduino IDE)

El IDE de Arduino es el software usado para escribir y cargar programas (sketches) a la placa. Incluye editor de código, compilador integrado, monitor serial para depuración, librería de ejemplos y gestor de librerías externas.

// Sección 03
Estructura de un Programa
3
Las dos funciones obligatorias

Todo programa de Arduino se compone de dos funciones principales que son obligatorias. La función setup() se ejecuta una sola vez al iniciar, mientras que loop() se ejecuta continuamente.

📌 Nota Importante Estas dos funciones son el corazón de cualquier sketch de Arduino. Sin ellas el programa no compilará.
estructura.ino
Arduino / C++ 
// Función setup: Se ejecuta una vez al inicio
void setup() {
  // Configuración de pines y variables iniciales
  pinMode(13, OUTPUT);  // Configura el pin 13 como salida
  Serial.begin(9600);       // Inicia comunicación serial
}

// Función loop: Se ejecuta continuamente
void loop() {
  // Código que se repite infinitamente
  digitalWrite(13, HIGH);  // Enciende el LED
  delay(1000);              // Espera 1 segundo
  digitalWrite(13, LOW);   // Apaga el LED
  delay(1000);              // Espera 1 segundo
}

Pines y Funciones Básicas
4
Tipos de pines disponibles
Tipo de Pin Función Valores
Digitales Entrada/Salida digital (ON/OFF) HIGH (5V) o LOW (0V)
Analógicos Lectura de valores variables 0 a 1023 (0V a 5V)
PWM (~) Salida analógica simulada 0 a 255 (0% a 100%)
Alimentación Suministro de energía 5V, 3.3V, GND

Proyecto Ejemplo: Blink LED
5
El "Hola Mundo" de Arduino

Hacer parpadear un LED es el primer proyecto que todo estudiante de Arduino realiza. Enseña los conceptos fundamentales de salidas digitales y control de tiempo.

🎯 Objetivo Hacer que un LED conectado al pin 13 parpadee cada segundo, alternando entre encendido y apagado.
blink.ino
Arduino / C++ 
// Definimos el pin del LED
const int ledPin = 13;

void setup() {
  // Configuramos el pin como salida
  pinMode(ledPin, OUTPUT);
}

void loop() {
  // Encendemos el LED
  digitalWrite(ledPin, HIGH);
  delay(1000);  // Esperamos 1 segundo

  // Apagamos el LED
  digitalWrite(ledPin, LOW);
  delay(1000);  // Esperamos 1 segundo
}
Arduino / C++ · Proyecto Blink — el más básico y fundamental
💡 Explicación Este programa hace que el LED integrado en el pin 13 se encienda durante un segundo y se apague durante otro segundo, repitiéndose continuamente. Es el punto de partida perfecto para comenzar con Arduino.
