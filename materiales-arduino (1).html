Componentes Esenciales

Estos son los componentes básicos que todo maker de Arduino debería tener:

⚡

Resistencias

Limitan el flujo de corriente en un circuito. Esenciales para proteger LEDs y otros componentes sensibles.

Especificaciones
  • Valores comunes: 220Ω, 330Ω, 1kΩ, 10kΩ
  • Potencia: 1/4W o 1/2W
  • Tolerancia: ±5% (dorada)
  • Identificación por código de colores
PasivoBásico
💡

LEDs

Diodos emisores de luz, perfectos para indicadores visuales y proyectos de iluminación.

Especificaciones
  • Voltaje: 1.8V - 3.3V típico
  • Corriente: 20mA máximo
  • Colores: Rojo, verde, azul, amarillo, RGB
  • Polaridad: Ánodo (+) y Cátodo (-)
SalidaVisual
🔲

Protoboard (Breadboard)

Placa de pruebas que permite crear circuitos sin soldadura, ideal para prototipos y experimentos.

Características
  • 830 puntos de conexión (estándar)
  • Conexiones internas horizontales
  • Rieles de alimentación verticales
  • Reutilizable infinitas veces
HerramientaEsencial
🔌

Cables Jumper

Cables de conexión para interconectar componentes en la protoboard y con Arduino.

Tipos
  • Macho-Macho: Protoboard a protoboard
  • Macho-Hembra: Arduino a protoboard
  • Hembra-Hembra: Sensores a Arduino
  • Longitudes: 10cm, 20cm, 30cm
ConexiónBásico
🔘

Botones Pulsadores

Interruptores momentáneos que permiten crear interfaces de entrada para tus proyectos.

Especificaciones
  • Tipo: Push button (4 pines)
  • Contacto: Normalmente abierto (NO)
  • Requiere resistencia pull-up/down
  • Voltaje: 12V DC máximo
EntradaDigital
🎚️

Potenciómetro

Resistencia variable que permite controlar voltaje y crear interfaces de control analógico.

Especificaciones
  • Valores: 1kΩ, 10kΩ, 100kΩ
  • 3 terminales: VCC, GND, Señal
  • Rotación: 270° o 300°
  • Uso: Control de volumen, brillo, etc.
EntradaAnalógico

Sensores Populares

Sensores que expanden las capacidades de interacción de tus proyectos:

📡

Sensor Ultrasónico HC-SR04

Mide distancias usando ondas ultrasónicas. Ideal para detectar obstáculos y crear sistemas de medición.

Especificaciones
  • Rango: 2cm a 400cm
  • Precisión: ±3mm
  • Voltaje: 5V DC
  • Pines: VCC, Trig, Echo, GND
SensorDistancia
🌡️

Sensor DHT11

Sensor de temperatura y humedad digital. Perfecto para estaciones meteorológicas y control ambiental.

Especificaciones
  • Temperatura: 0°C a 50°C (±2°C)
  • Humedad: 20% a 90% (±5%)
  • Voltaje: 3.3V - 5V
  • Comunicación: Digital (1-wire)
SensorAmbiental
👤

Sensor PIR HC-SR501

Detector de movimiento infrarrojo pasivo. Detecta presencia humana para sistemas de seguridad y automatización.

Especificaciones
  • Rango: 3m a 7m (ajustable)
  • Ángulo: 120° aproximadamente
  • Voltaje: 5V - 20V DC
  • Salida: Digital (HIGH/LOW)
SensorMovimiento
☀️

Fotoresistencia (LDR)

Sensor de luz que varía su resistencia según la intensidad lumínica. Ideal para proyectos de automatización con luz.

Especificaciones
  • Resistencia en oscuridad: 1MΩ
  • Resistencia en luz: 1kΩ - 10kΩ
  • Tiempo de respuesta: 20ms
  • Requiere divisor de voltaje
SensorLuz

Actuadores y Salidas

Componentes que permiten a Arduino interactuar con el mundo físico:

⚙️

Servomotor SG90

Motor de posición controlada con precisión. Ideal para brazos robóticos, mecanismos de dirección y más.

Especificaciones
  • Rotación: 180° (0° a 180°)
  • Torque: 1.8kg/cm a 5V
  • Velocidad: 0.1s/60° a 4.8V
  • Control: Señal PWM
ActuadorMovimiento
🔄

Motor DC

Motor de corriente continua para movimiento rotativo. Usado en robots móviles, ventiladores y más.

Especificaciones
  • Voltaje: 3V - 12V típico
  • Requiere driver L298N o similar
  • Control de velocidad con PWM
  • Control de dirección con H-bridge
ActuadorMotor
🔊

Buzzer Piezoeléctrico

Generador de tonos para alertas sonoras, alarmas y música simple. Disponible en activo y pasivo.

Especificaciones
  • Voltaje: 3V - 12V
  • Activo: Tono fijo al energizar
  • Pasivo: Requiere señal PWM
  • Frecuencia: 2kHz - 4kHz típico
SalidaAudio
🔌

Módulo Relé

Interruptor electrónico que permite controlar cargas de alto voltaje (AC/DC) de forma segura.

Especificaciones
  • Control: 5V DC (Arduino)
  • Carga: 110V/220V AC hasta 10A
  • Canales: 1, 2, 4, 8 relés
  • Incluye LED indicador
ActuadorPotencia

Pantallas y Displays

Elementos visuales para mostrar información:

📟

LCD 16x2

Pantalla de cristal líquido que muestra 2 filas de 16 caracteres. Ideal para mostrar mensajes y datos.

Especificaciones
  • Tamaño: 16 columnas x 2 filas
  • Con o sin módulo I2C
  • Voltaje: 5V
  • Retroiluminación LED azul/verde
DisplayTexto
🖥️

Display OLED 0.96"

Pantalla OLED pequeña con excelente contraste. Perfecta para interfaces compactas y wearables.

Especificaciones
  • Resolución: 128x64 píxeles
  • Tamaño: 0.96 pulgadas
  • Comunicación: I2C o SPI
  • Colores: Monocromático (blanco/azul)
DisplayGráfico
🔢

Display 7 Segmentos

Display LED para mostrar números. Ideal para relojes, contadores y medidores simples.

Especificaciones
  • Dígitos: 1 a 4 displays
  • Tipo: Ánodo o Cátodo común
  • Color: Rojo, verde, azul
  • Control: Directo o con MAX7219
DisplayNumérico
⬜

Matriz LED 8x8

Arreglo de 64 LEDs en formato matricial. Perfecto para crear animaciones, juegos y mensajes dinámicos.

Especificaciones
  • Tamaño: 8x8 LEDs (64 total)
  • Con controlador MAX7219
  • Encadenable (múltiples matrices)
  • Control SPI (3 pines)
DisplayGráfico

Consejos de Seguridad
🔴 Importante — Manejo Seguro de Componentes Siempre verifica la polaridad de componentes como LEDs, capacitores electrolíticos y diodos antes de conectarlos. Una conexión incorrecta puede dañar permanentemente el componente o tu Arduino.
⚡ Precaución con Voltajes Nunca conectes cargas que requieran más de 40mA directamente a los pines de Arduino. Usa transistores o relés para controlar motores y otros dispositivos de alta corriente.
🔋 Alimentación Adecuada No alimentes motores o servos desde los pines de 5V del Arduino. Usa una fuente de alimentación externa para evitar reiniciar el microcontrolador por caída de voltaje.
✓
Mejores Prácticas
Desconecta la alimentación antes de modificar conexiones
Usa resistencias limitadoras con LEDs
Verifica el código antes de cargar al Arduino
Mantén tu espacio de trabajo limpio y organizado
Documenta tus conexiones y código
